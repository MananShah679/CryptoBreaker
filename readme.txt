Installation Steps 

1. Create virtual environment
python -m venv venv

2. Activate it

Windows:

.\venv\Scripts\Activate.ps1

3. Install dependencies
pip install -r requirements.txt

Running the Project
Option 1: Terminal Mode (CLI)
python bruteforce_ciphers.py


This runs the classical command-line version.
